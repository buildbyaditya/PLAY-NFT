PLAY NFT 🎨

Live At : https://buildbyaditya.github.io/PLAY-NFT/


PLAY NFT is a beautifully designed, multi-page digital marketplace for crypto collectibles and non-fungible tokens (NFTs). It allows users to discover, collect, and sell digital art.

The standout feature of this project is that it is built entirely using HTML5 and CSS3 — absolutely Zero JavaScript (Zero JS). All animations, scrolling effects, and visual dynamics are handled purely through CSS.

🚀 Features
Zero JS Architecture: Fully functional UI and animations achieved strictly through HTML and CSS.

Modern Glassmorphism Design: Uses sleek, frosted-glass effects (glassmorphism classes) and glowing cards to create a futuristic Web3 aesthetic.

CSS-Only Animations: Continuous logo sliders, scrolling text animations, and hover effects built using CSS keyframes and transitions.

Multi-Page Layout: A fully linked structure connecting users to different sections of the platform.

Custom UI Elements: Gradient glowing borders, custom hover underlines for navigation, and customized buttons.

Responsive Typography: Utilizes the Poppins font family for a clean, modern, and readable interface.

🛠️ Tech Stack
HTML5: Semantic structuring and multi-page routing.

CSS3: Styling, layout (Flexbox/Grid), animations, glassmorphism, and gradient effects.

(No JavaScript was used in the making of this template)

📂 Project Structure
This is a multi-page website. Based on the navigation, the project consists of the following core HTML files:

nft.html - The Home page featuring the hero section, popular NFTs, top sellers, and categories. (Provided in the code above)

explore.html - Page dedicated to browsing various NFT collections.

market.html - The main marketplace for active auctions and listings.

artist.html - Directory of featured artists and creators.

news.html - Platform updates and Web3 news.

auth.html - The Registration/Login page for users to join the platform.

nft.css - The global stylesheet powering the entire website's design and animations.

💻 Getting Started
Since this project uses no external frameworks or JavaScript dependencies, running it is incredibly simple:

Clone or Download the repository to your local machine.

Ensure all your image assets (e.g., generated-image-*.png, search.png, etc.) are placed in the correct directories as referenced in the HTML file (e.g., root folder and /img/ folder).

Ensure your nft.css file is in the same directory as your HTML files.

Run the project by simply double-clicking nft.html (or your root index file) to open it in any modern web browser. No local server is required.

🎨 UI/UX Highlights
Hero Section: Displays a high-value NFT auction with a countdown, current bid, and dynamic user avatars.

Infinite Sliders: Partner logos and Top Seller profiles utilize infinite scrolling tracks to save space and add movement to the page.

Popular This Week: A grid/slider of highly rated NFTs complete with artist attribution, pricing in ETH, and like counts.

Categorized Exploration: Clean visual cards for categories like 3D Art, Abstract, Game, and Graffiti.
